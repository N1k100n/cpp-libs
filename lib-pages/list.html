List library (C++11)

The <list> header provides a way to create doubly linked lists in C++. The std::list class template is a container that encapsulates a doubly linked list of elements of a specific type. It provides efficient insertion and deletion of elements at any position in the list, but it does not provide random access to elements.

Key features of std::list

  • Doubly linked: The elements of a std::list are stored in a doubly linked list, which means that each element has pointers to the previous and next elements in the list.
  • Efficient insertion and deletion: Insertion and deletion of elements at any position in a std::list is efficient, as it only requires updating the pointers of the adjacent elements.
  • No random access: std::list does not provide random access to elements, as it is not possible to directly access an element at a specific index. You have to iterate through the list to reach the desired element.
  • Dynamic size: The size of a std::list can be changed at runtime by inserting or deleting elements.

How to use <list>

To use <list>, you need to include the <list> header file. You can then create a std::list object by specifying the type of the elements.

Example: Creating and manipulating a list

#include <iostream>
#include <list>

int main() {
    std::list<int> lst = {1, 2, 3, 4, 5}; // Create a list of integers

    lst.push_front(0); // Insert an element at the beginning
    lst.push_back(6); // Insert an element at the end

    for (int x : lst) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 0 1 2 3 4 5 6

    lst.remove(3); // Remove all elements with the value 3

    for (int x : lst) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 0 1 2 4 5 6

    return 0;
}

Example: Using iterators

#include <iostream>
#include <list>
#include <algorithm>

int main() {
    std::list<int> lst = {5, 2, 4, 1, 3};

    lst.sort(); // Sort the list using std::sort (efficient for linked lists)

    for (int x : lst) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 1 2 3 4 5

    return 0;
}

When to use <list>

<list> is useful when you need a container that supports efficient insertion and deletion of elements at any position. It is often used when you need to maintain a list of elements in a specific order and you need to be able to add or remove elements from the list frequently.

If you need a container that supports random access to elements, you should use std::vector instead. If you need a container that provides fast insertion and deletion at the beginning and end of the sequence, you should use std::deque instead.
