Format library (C++20)

The <format> header provides a type-safe and extensible way to format objects into strings. It replaces the older printf-style formatting with a more modern and flexible approach.

Key features of std::format

  • Type safety: std::format checks the types of the arguments at compile time, preventing many common formatting errors.
  • Extensibility: You can define custom formatters for your own types, allowing you to format them in a way that makes sense for your application.
  • Localization: std::format supports localization, so you can format numbers and dates in a way that is appropriate for the user's locale.
  • Performance: std::format is often faster than printf-style formatting, especially for complex formats.

How to use <format>

To use <format>, you need to include the <format> header file. You can then use the std::format function to format objects into strings.

Example: Basic formatting

#include <iostream>
#include <format>

int main() {
    std::string message = std::format("Hello, {}! The answer is {}.", "world", 42);
    std::cout << message << std::endl; // Output: Hello, world! The answer is 42.

    return 0;
}

Example: Custom formatting

#include <iostream>
#include <format>

struct Point {
    int x;
    int y;
};

template <>
struct std::formatter<Point> {
    template <typename FormatContext>
    auto format(const Point& p, FormatContext& ctx) {
        return std::format_to(ctx.out(), "({}, {})", p.x, p.y);
    }
};

int main() {
    Point p{10, 20};
    std::string message = std::format("The point is {}.", p);
    std::cout << message << std::endl; // Output: The point is (10, 20).

    return 0;
}

When to use <format>

<format> is useful whenever you need to format objects into strings. It is a powerful and flexible tool that can be used for a wide variety of formatting tasks. In most cases, it is a better choice than printf-style formatting, as it is type-safe and extensible.

It's important to be aware of the different format specifiers that are available. You can use these specifiers to control the way that numbers, dates, and other objects are formatted. You can also define your own custom formatters for your own types.
