Array library (C++11)

The <array> header provides a way to create fixed-size arrays in C++. The std::array class template is a container that encapsulates a fixed-size array of elements of a specific type. It provides similar functionality to C-style arrays, but with the added benefits of being a class type, such as providing member functions and supporting iterators.

Key features of std::array

  • Fixed size: The size of a std::array is fixed at compile time and cannot be changed.
  • Contiguous storage: The elements of a std::array are stored in contiguous memory locations, just like C-style arrays. This makes it efficient to access elements using pointers.
  • Class type: std::array is a class type, which means it can be used with other standard library components, such as algorithms and containers.
  • Bounds checking: std::array provides bounds checking through the at() method, which throws an exception if you try to access an element outside the valid range.

How to use <array>

To use <array>, you need to include the <array> header file. You can then create a std::array object by specifying the type of the elements and the size of the array.

Example: Creating and accessing an array

#include <iostream>
#include <array>

int main() {
    std::array<int, 5> arr = {1, 2, 3, 4, 5}; // Create an array of 5 integers

    std::cout << arr[0] << std::endl; // Access the first element (no bounds checking)
    std::cout << arr.at(2) << std::endl; // Access the third element (with bounds checking)

    try {
        std::cout << arr.at(10) << std::endl; // Access an element out of range
    } catch (const std::out_of_range& e) {
        std::cerr << "Error: " << e.what() << std::endl;
    }

    return 0;
}

Example: Using iterators

#include <iostream>
#include <array>
#include <algorithm>

int main() {
    std::array<int, 5> arr = {5, 2, 4, 1, 3};

    std::sort(arr.begin(), arr.end()); // Sort the array using std::sort

    for (int x : arr) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 1 2 3 4 5

    return 0;
}

When to use <array>

<array> is useful when you need a fixed-size array of elements and you want the benefits of a class type, such as member functions and iterators. It is often used when you know the size of the array at compile time and you don't need to resize it later.

If you need a dynamic-size array, you should use std::vector instead. If you need a fixed-size array and you don't need the benefits of a class type, you can use a C-style array.
