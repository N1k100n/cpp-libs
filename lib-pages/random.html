Random number generation library (C++11)

The <random> header provides a way to generate random numbers in C++. It defines a number of random number generators, which are objects that produce a sequence of pseudo-random numbers, as well as a number of random number distributions, which are objects that transform the output of a random number generator to follow a specific statistical distribution.

Key components in <random>

  • Random number generators: Classes that generate pseudo-random numbers, such as std::mt19937 (Mersenne Twister engine), std::linear_congruential_engine, and std::subtract_with_carry_engine.
  • Random number distributions: Classes that generate random numbers according to a specific statistical distribution, such as std::uniform_int_distribution (for integers), std::uniform_real_distribution (for floating-point numbers), std::normal_distribution (for normal distribution), and std::poisson_distribution (for Poisson distribution).
  • Other utilities: Classes and functions for seeding generators, generating non-deterministic random numbers, etc.

How to use <random>

To use <random>, you need to include the <random> header file. You can then create a random number generator object and a random number distribution object. You can then use the distribution object to generate random numbers from the generator.

Example: Generating random integers

#include <iostream>
#include <random>

int main() {
    std::random_device rd; // Obtain a random seed from the OS
    std::mt19937 gen(rd()); // Standard mersenne_twister_engine seeded with rd()
    std::uniform_int_distribution<> distrib(1, 10); // Define range for distribution

    for (int i = 0; i < 10; ++i) {
        std::cout << distrib(gen) << " "; // Generate and print a random number between 1 and 10
    }
    std::cout << std::endl;

    return 0;
}

Example: Generating random floating-point numbers

#include <iostream>
#include <random>

int main() {
    std::random_device rd; // Obtain a random seed from the OS
    std::mt19937 gen(rd()); // Standard mersenne_twister_engine seeded with rd()
    std::uniform_real_distribution<> distrib(0.0, 1.0); // Define range for distribution

    for (int i = 0; i < 10; ++i) {
        std::cout << distrib(gen) << " "; // Generate and print a random number between 0 and 1
    }
    std::cout << std::endl;

    return 0;
}

When to use <random>

<random> is useful when you need to generate random numbers in C++, such as in simulations, games, or cryptographic applications. It provides a number of tools that can be used to generate random numbers that follow a specific statistical distribution.

It's important to choose the appropriate random number generator and distribution for your needs. For most purposes, std::mt19937 is a good choice for a random number generator, and std::uniform_int_distribution and std::uniform_real_distribution are good choices for uniform distributions of integers and floating-point numbers, respectively.
