Iterator library

The <iterator> header provides a number of tools for working with iterators in C++. Iterators are objects that can be used to traverse containers and access their elements. The <iterator> header defines a number of iterator tags, which are used to indicate the capabilities of an iterator, as well as a number of functions for working with iterators.

Key concepts and components in <iterator>

  • Iterator tags: Enumerations that indicate the category of an iterator, such as std::input_iterator_tag, std::output_iterator_tag, std::forward_iterator_tag, std::bidirectional_iterator_tag, and std::random_access_iterator_tag. These tags are used to determine which operations are supported by an iterator.
  • Iterator adaptors: Function templates that create new iterators by modifying existing ones, such as std::reverse_iterator, std::move_iterator, std::inserter, std::back_inserter, and std::front_inserter.
  • Iterator functions: Functions for working with iterators, such as std::advance, std::distance, and std::next.

How to use <iterator>

To use <iterator>, you need to include the <iterator> header file. You can then use the iterator tags, adaptors, and functions provided by the library.

Example: Using std::reverse_iterator

#include <iostream>
#include <iterator>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};
    std::reverse_iterator<std::vector<int>::iterator> rit(v.end());
    std::reverse_iterator<std::vector<int>::iterator> rend(v.begin());

    for (; rit != rend; ++rit) {
        std::cout << *rit << " ";
    }
    std::cout << std::endl; // Output: 5 4 3 2 1

    // Using std::for_each with reverse iterators
    std::for_each(rit, rend, [](int& n){ std::cout << n * 2 << " "; });
    std::cout << std::endl; // Output: 10 8 6 4 2

    return 0;
}

Example: Using std::inserter

#include <iostream>
#include <iterator>
#include <list>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> v = {1, 2, 3};
    std::list<int> lst = {4, 5, 6};

    // Copy elements from lst to v at the beginning
    std::copy(lst.begin(), lst.end(), std::inserter(v, v.begin()));

    for (int x : v) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 4 5 6 1 2 3

    return 0;
}

When to use <iterator>

<iterator> is useful when you need to work with iterators in C++. It provides a number of tools that can make it easier to write generic code that can work with different types of iterators. It is often used in conjunction with the <algorithm> header, which provides a number of algorithms that operate on iterators.

It's important to understand the different iterator categories and to use the appropriate iterator adaptors and functions for your needs. For example, you should use std::reverse_iterator when you need to iterate over a container in reverse order, and you should use std::inserter when you need to insert elements into a container at a specific position.
