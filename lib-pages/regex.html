Regular expression library (C++11)

The <regex> header provides a way to work with regular expressions in C++. Regular expressions are a powerful tool for pattern matching and manipulation of strings. The <regex> header defines a number of classes and functions for creating, matching, and manipulating regular expressions.

Key components in <regex>

  • Regular expression objects: Objects of the std::regex class that represent regular expressions.
  • Match results: Objects of the std::match_results class that store the results of a regular expression match.
  • Regular expression algorithms: Functions for performing regular expression operations, such as std::regex_match (for matching a regular expression against a string), std::regex_search (for searching for a regular expression in a string), and std::regex_replace (for replacing parts of a string that match a regular expression).
  • Iterators: Objects of the std::regex_iterator and std::regex_token_iterator classes that can be used to iterate over the matches of a regular expression in a string.

How to use <regex>

To use <regex>, you need to include the <regex> header file. You can then create a std::regex object by specifying the regular expression pattern. You can then use the regular expression object to perform regular expression operations on strings.

Example: Matching a regular expression

#include <iostream>
#include <regex>
#include <string>

int main() {
    std::string str = "The quick brown fox jumps over the lazy dog.";
    std::regex rgx("quick\\s(\\w+)"); // Matches "quick" followed by a space and a word

    std::smatch match;
    if (std::regex_search(str, match, rgx)) {
        std::cout << "Match found: " << match[0] << std::endl; // Output: Match found: quick brown
        std::cout << "Capture group 1: " << match[1] << std::endl; // Output: Capture group 1: brown
    }

    return 0;
}

Example: Replacing parts of a string

#include <iostream>
#include <regex>
#include <string>

int main() {
    std::string str = "The quick brown fox jumps over the lazy dog.";
    std::regex rgx("fox");

    std::string new_str = std::regex_replace(str, rgx, "cat"); // Replace "fox" with "cat"

    std::cout << new_str << std::endl; // Output: The quick brown cat jumps over the lazy dog.

    return 0;
}

When to use <regex>

<regex> is useful when you need to perform complex pattern matching or manipulation of strings. It is often used in tasks such as validating input, searching for patterns in text, and replacing parts of strings. Regular expressions can be very powerful, but they can also be complex and difficult to understand. It's important to use them carefully and to test them thoroughly.

It's important to be aware of the different regular expression flags that are available, as well as the various escape sequences that can be used to match special characters. Some of the most commonly used components include std::regex, std::smatch, std::regex_match, std::regex_search, and std::regex_replace.
