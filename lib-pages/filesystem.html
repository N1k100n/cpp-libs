Filesystem library (C++17)

The <filesystem> header provides a way to work with files and directories in a portable way. It defines a number of classes and functions for performing file system operations, such as creating, deleting, renaming, and copying files and directories, as well as for querying file system information.

Key concepts and components in <filesystem>

  • Paths: Objects of the std::filesystem::path class that represent file and directory paths.
  • File status: Objects of the std::filesystem::file_status class that represent the status of a file or directory, such as whether it exists, whether it is a regular file, a directory, or a symbolic link.
  • Directory iterators: Objects of the std::filesystem::directory_iterator and std::filesystem::recursive_directory_iterator classes that can be used to iterate over the entries in a directory.
  • Functions: Functions for performing file system operations, such as std::filesystem::create_directory (for creating a directory), std::filesystem::remove (for deleting a file or directory), std::filesystem::rename (for renaming a file or directory), and std::filesystem::copy (for copying a file or directory).

How to use <filesystem>

To use <filesystem>, you need to include the <filesystem> header file. You can then create objects of the classes defined in <filesystem> and use their methods to perform file system operations.

Example: Working with paths

#include <iostream>
#include <filesystem>

int main() {
    std::filesystem::path p("./my_directory/my_file.txt"); // Create a path object

    std::cout << "Path: " << p << std::endl; // Output: Path: my_directory/my_file.txt
    std::cout << "Filename: " << p.filename() << std::endl; // Output: Filename: my_file.txt
    std::cout << "Parent directory: " << p.parent_path() << std::endl; // Output: Parent directory: my_directory

    return 0;
}

Example: Iterating over directory entries

#include <iostream>
#include <filesystem>

int main() {
    std::filesystem::path p("./my_directory"); // Create a path object for a directory

    for (const auto& entry : std::filesystem::directory_iterator(p)) { // Iterate over the directory entries
        std::cout << entry.path() << std::endl; // Print the path of each entry
    }

    return 0;
}

When to use <filesystem>

<filesystem> is useful when you need to perform file system operations in C++. It provides a number of tools that can be used to work with files and directories in a portable way. It is often used in conjunction with the <fstream> header, which provides a way to work with file streams.

It's important to be aware of the different classes and functions that are defined in <filesystem>, as well as the various error conditions that can occur during file system operations. Some of the most commonly used components include std::filesystem::path, std::filesystem::file_status, std::filesystem::directory_iterator, std::filesystem::create_directory, std::filesystem::remove, std::filesystem::rename, and std::filesystem::copy.
