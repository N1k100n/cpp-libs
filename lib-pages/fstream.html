File stream library

The <fstream> header provides a way to work with files in C++. It defines a number of classes for reading from and writing to files, as well as for performing other file-related operations. The classes in <fstream> are based on the iostream classes, and they inherit many of their features.

Key classes in <fstream>

  • std::ifstream: A class for reading from files.
  • std::ofstream: A class for writing to files.
  • std::fstream: A class for reading from and writing to files.

Key concepts and components in <fstream>

  • File modes: Flags that control how a file is opened, such as std::ios::in (for input), std::ios::out (for output), std::ios::app (for appending), and std::ios::binary (for binary mode).
  • File streams: Objects of the classes defined in <fstream> that are used to read from and write to files.
  • File operations: Functions for performing file-related operations, such as opening, closing, reading, writing, and seeking.

How to use <fstream>

To use <fstream>, you need to include the <fstream> header file. You can then create an object of one of the classes defined in <fstream> and use its methods to work with a file.

Example: Reading from a file

#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream ifs("myfile.txt"); // Create an input file stream object

    if (ifs.is_open()) {
        std::string line;
        while (std::getline(ifs, line)) { // Read the file line by line
            std::cout << line << std::endl; // Print each line to the console
        }
        ifs.close(); // Close the file
    } else {
        std::cerr << "Error: Unable to open file." << std::endl;
    }

    return 0;
}

Example: Writing to a file

#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ofstream ofs("myfile.txt"); // Create an output file stream object

    if (ofs.is_open()) {
        ofs << "This is a line of text." << std::endl; // Write a line of text to the file
        ofs << "This is another line of text." << std::endl; // Write another line of text to the file
        ofs.close(); // Close the file
    } else {
        std::cerr << "Error: Unable to open file." << std::endl;
    }

    return 0;
}

When to use <fstream>

<fstream> is useful when you need to read from or write to files in C++. It provides a number of tools that can be used to perform a wide range of file-related operations. It is often used in conjunction with the <iostream> header, which provides a number of classes and functions for working with input and output streams.

It's important to be aware of the different file modes that are available and to use the appropriate file mode for your needs. For example, you should use std::ios::binary when you need to work with binary files, and you should use std::ios::app when you need to append to a file.
