String library

The <string> header provides a way to work with strings in C++. It defines a class template std::basic_string that can be specialized for different character types, such as char (for narrow strings) and wchar_t (for wide strings). The <string> header also provides a number of typedefs for commonly used string types, such as std::string (for narrow strings) and std::wstring (for wide strings).

Key features of std::string

  • Dynamic size: std::string objects can grow or shrink as needed.
  • Character manipulation: std::string provides a number of methods for manipulating strings, such as inserting, deleting, replacing, and searching for characters.
  • String operations: std::string supports various string operations, such as concatenation, comparison, and substring extraction.
  • Automatic memory management: std::string automatically manages the memory used to store the string data, so you don't have to worry about allocating or deallocating memory.

How to use <string>

To use <string>, you need to include the <string> header file. You can then create a std::string object and use its methods to work with the string data.

Example: Creating and manipulating strings

#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello"; // Create a string object
    std::string str2 = " world";
    std::string str3 = str1 + str2; // Concatenate two strings

    std::cout << str3 << std::endl; // Output: Hello world
    std::cout << str3.length() << std::endl; // Output: 11
    std::cout << str3.substr(0, 5) << std::endl; // Output: Hello

    return 0;
}

Example: Searching for substrings

#include <iostream>
#include <string>

int main() {
    std::string str = "This is a test string";
    size_t pos = str.find("test"); // Find the position of the substring "test"

    if (pos != std::string::npos) {
        std::cout << "Substring found at position " << pos << std::endl; // Output: Substring found at position 10
    }

    return 0;
}

When to use <string>

<string> is useful whenever you need to work with strings in C++. It is a powerful and flexible tool that can be used for a wide variety of string manipulation tasks. In most cases, it is a better choice than C-style strings, as it is safer and easier to use.

It's important to be aware of the different methods that are available for manipulating strings. Some of the most commonly used methods include length(), substr(), find(), replace(), and insert().
