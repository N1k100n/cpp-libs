Date and time utilities

C++ includes support for two types of time manipulation:

● The chrono library, a flexible collection of types that track time with varying degrees of precision (e.g. std::chrono::time_point).

● C-style date and time library (e.g. std::time).

Chrono library (C++11)

The chrono library was introduced in C++11 to provide a more type-safe and flexible way to work with time. It offers various classes and functions to represent and manipulate time points, durations, and clocks.

Key components

  • Clocks: Represent different types of clocks, such as system time, steady time, and high-resolution time.
  • Time points: Represent specific moments in time.
  • Durations: Represent intervals of time.

Examples

Getting current time

#include <iostream>
#include <chrono>

int main() {
    auto now = std::chrono::system_clock::now();
    auto time = std::chrono::system_clock::to_time_t(now);
    std::cout << std::ctime(&time) << std::endl;
    return 0;
}

Measuring execution time

#include <iostream>
#include <chrono>

int main() {
    auto start = std::chrono::high_resolution_clock::now();
    // Some code to be measured
    for (int i = 0; i < 1000000; ++i) {
        // Do something
    }
    auto end = std::chrono::high_resolution_clock::now();
    auto duration = std::chrono::duration_cast<std::chrono::milliseconds>(end - start);
    std::cout << "Execution time: " << duration.count() << "ms" << std::endl;
    return 0;
}

Additional features

  • Time zone support (C++20): The chrono library has been extended in C++20 to include comprehensive time zone support.
  • Calendar and date operations (C++20): C++20 also added classes and functions for working with calendars and dates.

The chrono library provides a powerful and versatile toolset for handling time-related tasks in C++. Its type safety and flexibility make it a preferred choice over the traditional C-style time functions.
