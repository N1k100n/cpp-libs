Algorithm library

The <algorithm> header provides a number of functions for working with ranges of elements in C++. These functions include searching, sorting, copying, transforming, and other operations. The algorithms in <algorithm> are designed to work with iterators, which are objects that can be used to traverse containers and access their elements.

Key categories of algorithms in <algorithm>

  • Non-modifying sequence operations: Algorithms that operate on a sequence of elements without modifying them, such as std::for_each, std::find, std::count, std::equal, and std::search.
  • Modifying sequence operations: Algorithms that modify a sequence of elements, such as std::copy, std::move, std::transform, std::replace, std::fill, std::generate, std::remove, std::unique, and std::reverse.
  • Sorting: Algorithms for sorting a sequence of elements, such as std::sort, std::stable_sort, and std::partial_sort.
  • Searching and related operations: Algorithms for searching for elements in a sequence, such as std::binary_search, std::lower_bound, and std::upper_bound.
  • Min/max operations: Algorithms for finding the minimum or maximum element in a sequence, such as std::min_element and std::max_element.
  • Comparison operations: Algorithms for comparing two sequences of elements, such as std::equal and std::lexicographical_compare.
  • Permutation operations: Algorithms for generating permutations of a sequence of elements, such as std::next_permutation and std::prev_permutation.

How to use <algorithm>

To use <algorithm>, you need to include the <algorithm> header file. You can then use the functions provided by the library.

Example: Using std::sort to sort a vector

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> v = {5, 2, 4, 1, 3};

    std::sort(v.begin(), v.end()); // Sort the vector in ascending order

    for (int x : v) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 1 2 3 4 5

    return 0;
}

Example: Using std::find to search for an element

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    auto it = std::find(v.begin(), v.end(), 3); // Find the element with value 3

    if (it != v.end()) {
        std::cout << "Element found at position " << std::distance(v.begin(), it) << std::endl;
        // Output: Element found at position 2
    }

    return 0;
}

When to use <algorithm>

<algorithm> is useful whenever you need to perform common operations on ranges of elements. It provides a number of functions that are efficient and well-tested, and it can help you write more concise and readable code.

It's important to be familiar with the different algorithms in <algorithm> and to choose the appropriate algorithm for your needs. For example, you should use std::sort when you need to sort a sequence of elements, and you should use std::find when you need to search for an element in a sequence.
