Output stream library

The <ostream> header defines the std::ostream class, which is the base class for all output stream classes in C++. It provides a number of methods for writing data to output streams, such as the standard output stream (std::cout), the standard error stream (std::cerr), and files. The ostream class is part of the iostream library, which is used for performing input and output operations in C++.

Key features of std::ostream

  • Formatted output: std::ostream provides a number of methods for formatting output, such as setting the width of an output field, setting the precision of floating-point numbers, and inserting manipulators.
  • Unformatted output: std::ostream also provides methods for writing unformatted data to output streams, such as writing raw characters or blocks of memory.
  • Error handling: std::ostream provides methods for checking the state of an output stream and for handling errors that may occur during output operations.
  • Extensibility: You can create your own custom output stream classes by inheriting from std::ostream.

How to use <ostream>

To use <ostream>, you need to include the <iostream> header file. You can then use the methods of the std::ostream class to write data to output streams.

Example: Writing formatted data to the standard output stream

#include <iostream>
#include <iomanip> // For manipulators like std::setw and std::setprecision

int main() {
    int age = 30;
    double height = 1.75;
    std::string name = "John";

    std::cout << "Name: " << std::setw(10) << name << std::endl; // Set width for name
    std::cout << "Age: " << age << std::endl;
    std::cout << "Height: " << std::fixed << std::setprecision(2) << height << std::endl; // Format height

    return 0;
}

Example: Writing unformatted data to the standard output stream

#include <iostream>

int main() {
    char buffer[] = "This is a string of characters.";
    std::cout.write(buffer, sizeof(buffer) - 1); // Write the string to the standard output stream
    std::cout << std::endl;

    return 0;
}

When to use <ostream>

<ostream> is useful whenever you need to write data to output streams in C++. It provides a number of tools that can be used to format output, handle errors, and perform other output-related operations. It is often used in conjunction with the <iostream> header, which provides a number of classes and objects for working with input and output streams.

It's important to be aware of the different methods that are available for writing data to output streams, as well as the various manipulators that can be used to format output. Some of the most commonly used methods include operator<< (for inserting data into the stream), put() (for writing a single character), write() (for writing a block of memory), and flush() (for flushing the output buffer).
