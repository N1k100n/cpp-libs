C standard library

The <cstdlib> header provides a number of general utility functions, including functions for dynamic memory management, random number generation, integer arithmetic, searching and sorting, environment control, integral arithmetics, and converting strings to numeric values.

Key functions and macros in <cstdlib>

  • Dynamic memory management:
    • malloc: Allocates a block of memory.
    • calloc: Allocates a block of memory and initializes it to zero.
    • realloc: Reallocates a block of memory.
    • free: Deallocates a block of memory.
  • Random number generation:
    • rand: Generates a pseudo-random integer.
    • srand: Seeds the pseudo-random number generator.
  • Integer arithmetic:
    • abs: Returns the absolute value of an integer.
    • div: Performs integer division and returns the quotient and remainder.
  • Searching and sorting:
    • bsearch: Performs a binary search in a sorted array.
    • qsort: Sorts an array using the quicksort algorithm.
  • Environment control:
    • getenv: Gets the value of an environment variable.
    • system: Executes a system command.
    • exit: Terminates the program.
  • String conversion:
    • atoi: Converts a string to an integer.
    • atol: Converts a string to a long integer.
    • atof: Converts a string to a double.

How to use <cstdlib>

To use <cstdlib>, you need to include the <cstdlib> header file. You can then use the functions and macros provided by the library in your code.

Example: Using malloc and free for dynamic memory allocation

#include <iostream>
#include <cstdlib>

int main() {
    int* arr = (int*)malloc(5 * sizeof(int)); // Allocate memory for 5 integers

    if (arr == nullptr) {
        std::cerr << "Memory allocation failed!" << std::endl;
        return 1;
    }

    for (int i = 0; i < 5; ++i) {
        arr[i] = i * 2;
    }

    for (int i = 0; i < 5; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl; // Output: 0 2 4 6 8

    free(arr); // Deallocate the memory

    return 0;
}

Example: Using rand and srand for random number generation

#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
    srand(time(0)); // Seed the random number generator with the current time

    for (int i = 0; i < 5; ++i) {
        std::cout << rand() % 100 << " "; // Generate a random number between 0 and 99
    }
    std::cout << std::endl;

    return 0;
}

When to use <cstdlib>

<cstdlib> is a fundamental library that is used in many C++ programs. The functions provided by this library are essential for performing various tasks, such as dynamic memory management, random number generation, and more. You should include <cstdlib> in any program that uses these functionalities.
