Initializer list library

The <initializer_list> header provides a way to create initializer lists, which are used to initialize containers and other objects with a list of values. Initializer lists are a convenient way to create collections of data without having to explicitly specify the type of each element.

Key features of std::initializer_list

  • Allows uniform initialization: std::initializer_list can be used to initialize objects of any type, including built-in types, user-defined types, and containers.
  • Supports variable number of elements: You can create initializer lists with any number of elements.
  • Provides access to elements: You can access the elements of an initializer list using iterators or range-based for loops.

How to use std::initializer_list

To use std::initializer_list, you need to include the <initializer_list> header file. You can then create an initializer list by enclosing a comma-separated list of values in curly braces.

Example: Initializing a vector with an initializer list

#include <iostream>
#include <initializer_list>
#include <vector>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5}; // Initialize a vector with an initializer list

    for (int x : v) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 1 2 3 4 5

    return 0;
}

Example: Initializing a custom class with an initializer list

#include <iostream>
#include <initializer_list>

class MyClass {
public:
    MyClass(std::initializer_list<int> list) {
        for (int x : list) {
            data.push_back(x);
        }
    }

    void print() {
        for (int x : data) {
            std::cout << x << " ";
        }
        std::cout << std::endl;
    }

private:
    std::vector<int> data;
};

int main() {
    MyClass obj = {10, 20, 30}; // Initialize a custom class with an initializer list
    obj.print(); // Output: 10 20 30

    return 0;
}

When to use std::initializer_list

std::initializer_list is useful when you need to create collections of data with a variable number of elements. It is often used to initialize containers, but it can also be used to initialize other types of objects.

Initializer lists are a convenient way to create collections of data without having to explicitly specify the type of each element. This can make your code more concise and readable.
