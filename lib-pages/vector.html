Vector library (C++11)

The <vector> header provides a way to create dynamic arrays in C++. The std::vector class template is a container that encapsulates a dynamic array of elements of a specific type. It provides efficient access to elements using indices, and it can grow or shrink as needed.

Key features of std::vector

  • Dynamic size: The size of a std::vector can be changed at runtime by inserting or deleting elements.
  • Random access: You can access any element in a std::vector directly using its index, just like with C-style arrays. This makes it very efficient to access elements.
  • Contiguous storage: The elements of a std::vector are stored in contiguous memory locations, which makes it efficient to iterate through the elements and to pass the data to C-style APIs.
  • Automatic memory management: std::vector automatically manages the memory used to store the elements, so you don't have to worry about allocating or deallocating memory yourself.

How to use <vector>

To use <vector>, you need to include the <vector> header file. You can then create a std::vector object by specifying the type of the elements.

Example: Creating and manipulating a vector

#include <iostream>
#include <vector>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5}; // Create a vector of integers

    v.push_back(6); // Add an element to the end
    v.pop_back(); // Remove the last element

    for (int x : v) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 1 2 3 4 5

    std::cout << v[2] << std::endl; // Access the third element (no bounds checking)
    std::cout << v.at(2) << std::endl; // Access the third element (with bounds checking)

    return 0;
}

Example: Using iterators

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> v = {5, 2, 4, 1, 3};

    std::sort(v.begin(), v.end()); // Sort the vector using std::sort

    for (int x : v) {
        std::cout << x << " ";
    }
    std::cout << std::endl; // Output: 1 2 3 4 5

    return 0;
}

When to use <vector>

<vector> is the most commonly used container in C++. It is useful when you need a dynamic array that can grow or shrink as needed, and when you need efficient access to elements using indices. It is often used as a default container when you don't have specific requirements for another container.

If you need a container that supports efficient insertion and deletion of elements at any position, you should use std::list or std::deque instead. If you need a fixed-size array, you should use std::array instead.
